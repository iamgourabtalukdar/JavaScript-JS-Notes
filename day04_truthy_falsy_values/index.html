Truthy & Falsy Values

  1. Truthy Values :
    • number: [3, -3, 3.4, -3.4, Infinity, -Infinity]
    • string: [all string with length > 0 ---OR--- all string except empty string]
    • Boolean: [true]
    • bigint: [all values except 0n]
  2. Falsy Values :
    • number: [0, NaN]
    • string: [Empty string]
    • Boolean: [false]
    • undefined: [undefined]
    • null: [null]
    • bigint: [0n]
